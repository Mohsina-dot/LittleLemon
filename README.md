Little Lemon Restaurant API - Peer Review Guide

Please use the following API paths to test the application using Thunder Client, Insomnia or a browser.

1. User Authentication & Registration (Djoser):
   - User Registration: /auth/users/
   - Token Login: /auth/token/login/
   - Token Logout: /auth/token/logout/

2. Menu API Endpoints:
   - View all menu items / Add new item: /restaurant/menu/
   - View, Update or Delete a single item: /restaurant/menu/<id>

3. Booking API Endpoints:
   - View all bookings / Create a booking: /restaurant/booking/
   - View, Update or Delete a single booking: /restaurant/booking/<id>
